# 江湖残卷 - 卡牌Roguelike Demo

## 游戏概述
武侠写实风格Roguelike竖屏小游戏，核心玩法为**卡牌构筑 + 回合制策略战斗**。
- 竖屏适配，移动端友好
- 4个职业，各有专属卡组
- 章节推进，每关3波战斗
- 局内选卡永久解锁
- 多种流派：连招流、暴击流、毒杀流、反击流、献祭流等

## 技术方案
- 前端：单HTML文件，原生JS + CSS
- 美术：Emoji图标（快速迭代）
- 部署：静态托管，即时预览

## 核心功能
- [x] 4职业系统（剑侠/刺客/武僧/萨满）
- [x] 卡牌系统（基础卡+解锁卡，50+张）
- [x] 回合制战斗（能量/抽牌/弃牌循环）
- [x] 敌人AI（普通怪+精英+BOSS）
- [x] 章节推进（5章节，每章3波）
- [x] 随机事件（商店/休息/江湖恩怨/宝箱）
- [x] 永久解锁（局内获得新卡永久解锁）
- [x] 数据存储（localStorage持久化）

## 文件结构
- `index.html` - 完整游戏（所有逻辑）
- `assets/` - 资源文件夹（预留）

## 运行方式
直接在浏览器打开index.html即可游玩
也可以部署到任意静态托管服务
